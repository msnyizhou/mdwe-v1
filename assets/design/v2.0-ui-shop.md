## 5. 支付页 `payment.html?order_id=20260516-001`

```
┌──────────────────────────────────────────────┐
│  💳 支付                                      │
├──────────────────────────────────────────────┤
│  📦 订单: 20260516-001                       │
│  商品: 期货交易心得 × 1                       │
│  卖家: 张三                                   │
│  金额: ¥29.90                                │
│                                              │
│  ┌──────────────────────────────┐            │
│  │                              │            │
│  │    [卖家收款码图片]           │            │
│  │                              │            │
│  │    微信 / 支付宝 扫码支付     │            │
│  │                              │            │
│  └──────────────────────────────┘            │
│                                              │
│  支付完成后，请点击下方按钮确认：              │
│  [我已支付]                                   │
│                                              │
│  ⏱ 订单 30 分钟后自动取消                     │
└──────────────────────────────────────────────┘
```

**支付页说明：**
- 平台商品 → 显示平台收款码
- 会员商品 → 显示卖家收款码
- 买家扫码付款后，点击「我已支付」
- 卖家在订单管理中确认收到款项后，订单变为 `paid`
- 这种模式不要求平台处理资金，降低合规门槛
